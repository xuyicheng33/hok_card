## 🎮 游戏简介

这是一款基于王者荣耀英雄的回合制卡牌对战游戏，支持在线联机对战。玩家可以通过创建或加入房间，与朋友进行刺激的3v3战斗。

**当前版本：** v1.0.0  
**在线服务器：** 121.199.78.133:3000

## 🎯 游戏特色

### 核心机制
- ✅ **回合制战斗** - 策略性强，思考时间充足
- ✅ **行动点系统** - 每回合3次行动机会
- ✅ **技能点系统** - 初始4点，上限6点，需合理分配
- ✅ **金币系统** - 购买装备，合成进阶装备
- ✅ **服务器权威** - 所有计算在服务器端，防止作弊

### 英雄系统
- 8个王者荣耀英雄：瑶、大乔、公孙离、澜、孙尚香、朵莉亚、少司缘、杨玉环
- 每个英雄拥有独特的主动技能和被动技能
- 固定3v3阵容：
  - **蓝方：** 瑶 + 大乔 + 公孙离
  - **红方：** 澜 + 孙尚香 + 朵莉亚

### 装备系统
- **基础装备**：7种（铁剑、搏击拳套、雷鸣刃、匕首、红玛瑙、布甲、提神水晶）
- **进阶装备**：9种（通过合成获得）
- **装备合成**：2件基础装备 + 10金币 → 1件进阶装备
- **特殊效果**：增伤、回血、暴击等

## 🏆 游戏亮点

### 战术深度
- 🎯 **暴击流** - 公孙离：堆暴击率，溢出转暴击效果
- ⚔️ **斩杀流** - 澜：叠攻击力，触发狩猎被动
- 💊 **续航流** - 朵莉亚：全队回血，持久战
- 💥 **爆发流** - 大乔：濒死反杀，AOE真伤

### 策略选择
- 优先发育还是直接进攻？
- 金币用于购买装备还是合成装备？
- 技能点何时使用？留给关键技能还是尽早释放？
- 装备给谁？输出英雄还是前排坦克？

## 🛠️ 技术架构

### 客户端
- **引擎：** Godot 4.5
- **语言：** GDScript
- **通信：** WebSocket

### 服务器端
- **运行环境：** Node.js
- **框架：** WebSocket (ws)
- **架构：** 服务器权威，所有计算在服务器端完成
- **部署：** 阿里云 121.199.78.133:3000

### 关键特性
- 服务器端权威计算（防作弊）
- 实时双向通信
- 房间系统
- 自动胜负判定
- 金币一致性校验

## ⚠️ 注意事项

### 游戏限制
- 房间ID为1-9的单位数（容易冲突）
- 每个房间只支持2个玩家
- 固定3v3阵容，暂不支持英雄选择
- 断线后无法重连，游戏结束

### 已知问题
- 少司缘被动"心之牵绊"暂未在当前阵容启用
- 杨玉环被动"霓裳风华"的持续效果未完全实现
