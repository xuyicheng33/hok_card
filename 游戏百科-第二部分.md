## 📖 装备系统完全指南

### 装备获取

#### 购买装备
**操作流程：**
1. 点击"💰购买装备(15)"按钮（15金币）
2. 系统随机抽取3个基础装备
3. 从3个中选择1个


**限制条件：**
- 只能在自己的回合操作
- 金币≥15
- 不消耗行动点

---

#### 装备物品
**操作流程：**
1. 购买后显示装备选择面板
2. 点击要装备的物品
3. 点击要装备的英雄
4. 装备立即生效

**装备限制：**
- 每个英雄最多2件装备
- 装备已满无法再装
- 效果立即生效

### 装备合成详解

#### 合成操作步骤
```
1. 确保英雄装备≥2件
2. 确保金币≥10
3. 确保是己方回合
4. 点击"🔨合成装备(10)"按钮
5. 系统高亮可合成英雄（橙色光圈）
6. 点击要合成的英雄
7. 系统自动选2个装备作为材料
8. 显示预览：材料1 + 材料2
9. 服务器验证配方
```

## 🎯 战斗机制详解

### 普通攻击计算

#### 完整伤害计算流程
```
步骤1：基础伤害（护甲减免已计算在内）
基础伤害 = 攻击者攻击力 × (200 / (目标护甲 + 200))
护甲减免率 = 目标护甲 / (目标护甲 + 200)

步骤2：增伤效果
增伤后伤害 = 基础伤害 × (1 + 增伤百分比)
增伤来源：
- 装备：匕首 +3%，穿云弓 +5%，速击之枪 +7%
- 被动：澜狩猎 +30%（目标血量<50%）

步骤3：暴击判定
if (随机数 < 暴击率):
    最终伤害 = 增伤后伤害 × 暴击效果
else:
    最终伤害 = 增伤后伤害

步骤4：闪避判定（仅对公孙离，仅能闪避普通攻击）
if (随机数 < 25%):
    闪避成功，实际伤害=0，流程结束

步骤5：护盾吸收（真实伤害直接扣除生命值，且不会受护甲减免）
if (目标护盾 > 0):
    护盾剩余 = 护盾 - 实际伤害
    if (护盾剩余 < 0):
        生命值 -= |护盾剩余|
        护盾 = 0
    else:
        护盾 = 护盾剩余
else:
    生命值 -= 实际伤害
```

#### 伤害计算示例
```
案例：澜（攻击500+日冕25=525）攻击公孙离（护甲150，护盾0）

1. 基础伤害 = 525 × (200/(150+200)) = 525 × 0.5714 = 300
   护甲减免率 = 150/350 = 42.86%
   
2. 公孙离血量40%，触发澜被动：300 × 1.3 = 390

3. 暴击判定（10%概率）：
   - 暴击：390 × 1.3 = 507
   - 不暴击：390
   
4. 闪避判定（25%概率）：
   - 闪避：伤害=0
   - 不闪避：继续
   
5. 无护盾，直接扣血：507点伤害（暴击情况）
```
