## 📖 装备系统完全指南

### 装备获取

#### 购买装备
**操作流程：**
1. 点击"💰购买装备(15)"按钮（15金币）
2. 系统随机抽取3个基础装备
3. 从3个中选择1个
4. 装备进入背包

**限制条件：**
- 只能在自己的回合操作
- 金币≥15
- 不消耗行动点

---

#### 装备物品
**操作流程：**
1. 购买后显示装备选择面板
2. 点击要装备的物品
3. 点击要装备的英雄
4. 装备立即生效

**装备限制：**
- 每个英雄最多2件装备
- 装备已满无法再装
- 效果立即生效

## 🎯 战斗机制详解

### 普通攻击计算

#### 完整伤害计算流程
```
步骤1：基础伤害（护甲减免已计算在内）
基础伤害 = 攻击者攻击力 × (200 / (目标护甲 + 200))
护甲减免率 = 目标护甲 / (目标护甲 + 200)

步骤2：增伤效果
增伤后伤害 = 基础伤害 × (1 + 增伤百分比)
增伤来源：
- 装备：匕首 +3%，穿云弓 +5%，速击之枪 +7%
- 被动：澜狩猎 +30%（目标血量<50%）

步骤3：暴击判定
if (随机数 < 暴击率):
    最终伤害 = 增伤后伤害 × 暴击效果
else:
    最终伤害 = 增伤后伤害

步骤4：闪避判定（仅对公孙离）
if (随机数 < 25%):
    闪避成功，实际伤害=0，流程结束

步骤5：护盾吸收
if (目标护盾 > 0):
    护盾剩余 = 护盾 - 实际伤害
    if (护盾剩余 < 0):
        生命值 -= |护盾剩余|
        护盾 = 0
    else:
        护盾 = 护盾剩余
else:
    生命值 -= 实际伤害
```

#### 伤害计算示例
```
案例：澜（攻击500+日冕25=525）攻击公孙离（护甲150，护盾0）

1. 基础伤害 = 525 × (200/(150+200)) = 525 × 0.5714 = 300
   护甲减免率 = 150/350 = 42.86%
   
2. 公孙离血量40%，触发澜被动：300 × 1.3 = 390

3. 暴击判定（10%概率）：
   - 暴击：390 × 1.3 = 507
   - 不暴击：390
   
4. 闪避判定（25%概率）：
   - 闪避：伤害=0
   - 不闪避：继续
   
5. 无护盾，直接扣血：507点伤害（暴击情况）
```

## 💡 游戏策略指南

### 开局策略

#### 蓝方（瑶+大乔+公孙离）
**核心思路：** 保护公孙离输出，利用瑶和大乔提供保护

**前3回合：**
1. 优先用公孙离攻击敌方澜（威胁最大）
2. 瑶给公孙离套盾+加暴击
3. 积攒金币购买装备（目标：给公孙离买暴击装备）

**装备推荐：**
- 公孙离：搏击拳套、雷鸣刃 → 狂暴双刃
- 瑶：红玛瑙、提神水晶 → 熔炼之心
- 大乔：红玛瑙、布甲 → 守护者之铠

---

#### 红方（澜+孙尚香+朵莉亚）
**核心思路：** 澜叠攻击力，孙尚香破甲，朵莉亚持续治疗

**前3回合：**
1. 澜使用技能叠攻击（2点技能 → 永久+100攻击）
2. 朵莉亚持续治疗澜
3. 孙尚香点杀敌方残血英雄

**装备推荐：**
- 澜：铁剑、匕首 → 速击之枪
- 孙尚香：搏击拳套、匕首 → 穿云弓
- 朵莉亚：红玛瑙、提神水晶 → 熔炼之心

## 📊 高级技巧

### 伤害最大化
**暴击流公孙离：**
```
装备路线：
1. 购买搏击拳套（暴击率30%）
```

## ⚠️ 常见错误

### 操作失误
1. ❌ **非己方回合操作** → 所有操作被服务器拒绝
2. ❌ **行动点用完还想攻击** → 按钮禁用
3. ❌ **技能点不足释放技能** → 提示技能点不足
4. ❌ **金币不足购买/合成** → 按钮禁用

### 战术失误
1. ❌ **过早使用大乔大招** → 浪费4技能点
2. ❌ **不给核心输出套盾** → 被集火秒杀
3. ❌ **忽视装备合成** → 后期属性差距大
4. ❌ **技能点全用完** → 无法应对突发情况

### 装备失误
1. ❌ **给坦克买攻击装备** → 浪费
2. ❌ **不合成进阶装备** → 性价比低
3. ❌ **装备槽满了还买** → 无法装备新装备
