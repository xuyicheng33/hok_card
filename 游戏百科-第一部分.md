## 游戏概述

### 基本信息
- **游戏类型：** 回合制卡牌对战
- **对战模式：** 3v3 在线联机
- **3V3阵容：**
  - 双方各自选择3名英雄：
  选择顺序如下：
  蓝方选择一名英雄，红方选择两名英雄，蓝方再选择两名英雄，红方最后选择一名英雄。

### 胜利条件
消灭对方所有英雄，系统自动判定胜负

## 游戏术语

### 基础属性
- **生命值（Health）：** 英雄的当前血量，降至0时死亡
- **最大生命值（Max Health）：** 英雄的生命值上限
- **攻击力（Attack）：** 影响普攻和技能伤害
- **护甲（Armor）：** 减少受到的物理伤害
- **护盾（Shield）：** 吸收伤害的临时保护层，优先于生命值消耗

### 暴击系统
- **暴击率（Crit Rate）：** 触发暴击的概率（0-100%）
- **暴击效果（Crit Damage）：** 暴击时的伤害倍率（默认130%，上限200%）

### 特殊属性
- **闪避率（Dodge Rate）：** 完全闪避攻击的概率（公孙离25%）
- **增伤（Damage Amplify）：** 提升造成伤害的百分比
- **每回合恢复（Heal Per Turn）：** 回合开始时自动恢复生命值

### 伤害类型
- **物理伤害：** 受护甲减免，公式：基础伤害 = 攻击力 × 200/(护甲+200)
- **真实伤害：** 无视护甲和护盾，直接扣除生命值

## 游戏流程

### 1. 房间创建与加入
**房主：** 点击"创建房间" → 获得1-9的房间ID → 告知朋友  
**客户端：** 点击"加入房间" → 输入房间ID → 匹配成功进入战斗

### 2. 战斗初始状态
- 技能点：4点 | 金币：10金币 | 行动点：3次/回合 | 回合：第1回合

### 3. 每回合流程
```
回合开始 →
  ├─ 装备回血触发（熔炼之心+50、提神水晶+30）
  ├─ 朵莉亚被动触发（全队+50生命）
  ├─ 技能点增加（第3回合起+1/回合）
  └─ 行动点重置为3

进行操作（消耗1次行动）→
  ├─ 普通攻击
  ├─ 释放技能（消耗技能点）
  ├─ 购买装备（15金币/次，不消耗行动点）
  ├─ 装备物品（不消耗行动点）
  └─ 合成装备（10金币，不消耗行动点）

结束回合 → 对手回合 → 循环

一方全灭 → 显示战斗结果 → 房间清理
```

## 核心系统

### 回合系统
- **奇数回合（1,3,5...）：** 房主（蓝方）
- **偶数回合（2,4,6...）：** 客户端（红方）
- **服务器权威：** 防止作弊，只能在己方回合操作

### 行动点系统
- **每回合3次行动点**
- **消耗规则：** 普攻/技能消耗1次，购买/装备/合成不消耗
- **重置时机：** 回合结束时重置为3

### 技能点系统
- **初始：** 4点 | **上限：** 6点
- **获取方式：**
  1. 第3回合起每回合+1点
  2. 孙尚香被动：击杀敌人70%概率+1
  3. 大乔被动：濒死时+3（溢出转护盾，1点=150护盾）
  4. 少司缘被动：回合切换窃取对手1点

### 金币系统
- **初始：** 10金币
- **回合收入：** 第1回合0，第2回合+1，第3-4回合+2，第5回合起+3
- **消耗：** 购买装备15金币/次，合成装备10金币/次
